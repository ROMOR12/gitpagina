Clone:

Con Clone puedes descargar un repositorio usando su URL. En consola se ejecuta con el comando: git clone "URL"


Hecho por Roberto Ortuño

Init:

Con init puedes convertir una carpeta en un repositorio de Git. En consola se ejecuta con el comando: git init


Hecho por Roberto Ortuño

Add:

Con add puedes añadir archivos específicos o todos al área de staging. En consola se ejecuta con el comando: git add "archivo"


Hecho por Roberto Ortuño

Commit:

Con commit puedes guardar los cambios del área de staging en el historial del repositorio. En consola se ejecuta con el comando: git commit -m "mensaje"


Hecho por Roberto Ortuño

Log:

Con log puedes monstrar el historial de commits. En consola se ejecuta con el comando: git log


Hecho por Roberto Ortuño

Diff:

Con status puedes monstrar los cambios entre el área de trabajo y el staging. En consola se ejecuta con el comando: git diff


Hecho por Roberto Ortuño

Pull:

Con status puedes descargar y combinar cambios desde un repositorio remoto. En consola se ejecuta con el comando: git pull


Hecho por Roberto Ortuño

Push:

Con status puedes subir los archivos locales al repositoro remoto. En consola se ejecuta con el comando: git push


Hecho por Roberto Ortuño
